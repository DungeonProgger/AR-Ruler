# AR-Ruler

Проект разработан в рамках курсовой работы (2 курс). Приложение предназначено для очков смешанной реальности **Microsoft HoloLens 2** и реализовано с использованием **Mixed Reality Toolkit (MRTK)**.

## Описание функциональности

Приложение позволяет пользователю производить измерения геометрических параметров физических объектов в реальном времени.

**Доступные режимы измерений:**
* **Прямоугольные объекты:** измерение длины, ширины и высоты.
* **Цилиндрические объекты:** измерение высоты и диаметра.
* **Произвольные измерения:**
    * Длина отрезка между двумя точками.
    * Вычисление объема фигуры.
    * Измерение углов.
